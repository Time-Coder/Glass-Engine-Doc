介绍

说明

Glass Engine 是一个 Python 原生的 3D 实时渲染引擎，免费开源。你可以使用它在你的 Python 程序界面的任何位置嵌入可交互的 3D 画面。与同类产品 Unreal Engine、Unity 3D、OSG、OGRE、Panda3D 等相比，Glass Engine 具有如下优势：

  • 更加轻量小体积，pip install glass_engine 直接安装；

  • 更加直观易用，开发逻辑和编程接口更加符合人类直觉；

  • Python 原生，并不是对 C++ 程序的 Python 绑定，行为更加 Pythonic；

在渲染方面，Glass Engine 具有以下特色：

  • 采用 场景图 (Scene Graph) 形式组织场景，对场景进行逻辑分组、固结运动、设置相对坐标等操作更加方便；

  • 用户可控的实例化渲染 (Instanced Rendering)，极大程度的合并渲染批次，减少渲染调用；

  • 用户透明的顶点共享机制，使得相同参数几何体共用同一组顶点和索引，提高渲染效率；

  • 支持 自定义几何体 的动态构建，无需模型文件也能创建丰富场景；

  • 内置三十多种 基本几何体，对一些简单几何体的创建无需从模型加载也不需要自定义顶点索引；

  • 支持超过四十种模型文件格式的 模型 加载；

  • 支持多种 着色模型，包括 Flat、Gouraud、Phong、Phong-Blinn、Toon、Oren-Nayar、Minnaert、Fresnel、PBR 等；

  • 支持 多相机 多视口，多视口共享场景资源；

  • 用户透明的 Shader 预编译和增量编译过程，可极大程度提高程序启动速度；

  • 支持 前向渲染 (Forward Rendering) 和 延迟着色法 (Deferred Rendering) 管线，可由用户自由选择；

  • 使用基于物理的 泛光 技术 (Physical based Bloom)，提升泛光真实感；

  • 支持基于弥散圆 (Circle of Confusion， CoC) 理论的 景深 效果，并可根据点击自动渐变对焦；

  • 使用顺序无关的半透明 (Order Independent Transparent, OIT) 技术渲染半透明物体，用户不再需要关心渲染顺序；

  • 采用级联阴影映射 (Casecading Shadow Map, CSM) 技术渲染阴影，极大程度上提高了阴影清晰度；

  • 支持动态环境映射，更逼真的反射和折射效果，倒影中仍能呈现半透明、阴影等效果；

  • 提供可叠加的 后处理效果 机制，内置多种后处理效果，并可轻松 自定义后处理效果；

  • 支持 Shadertoy 动态纹理，Shadertoy 代码可直接转化为纹理贴图使用；

开发路线

Glass Engine 当前为 0.0.1 版本，后续将不断迭代升版，后续 10 个版本的开发计划为：

  • 0.0.1 版本：基本渲染架构（已完成）

    • ☑ 渲染基本要素：场景图、光源、相机、网格、简单材质系统、模型加载

    • ☑ 场景环境：天空盒、天空穹顶、雾

    • ☑ 后处理效果：SSAO、HDR、LUT、FXAA、景深、泛光、自定义后效

    • ☑ 常见渲染管线：前向渲染、延迟着色法、自定义管线

    • ☑ 键鼠交互模式：场景漫游、模型浏览、自定义模式

  • 0.0.2 版本：完善渲染要素

    • ☐ 高级软阴影算法：CSM、VSM、ESM、MSM、VSSM、EVSM

    • ☐ 半透明物体阴影、折射体阴影

    • ☐ 基于有符号距离场的文字渲染

    • ☐ IBL 以及 Specular/Glossiness 工作流的 PBR

    • ☐ BVH 树、视锥体剔除以及自动 Lod 等提高渲染效率的算法

    • ☐ USD 格式保存

  • 0.0.3 版本：完善渲染要素

    • ☐ 更多高级着色模型：清漆、次表面、皮肤、毛发、布料等，向 Unreal Engine 看齐

    • ☐ 自定义着色模型

    • ☐ 面光源

    • ☐ 粒子特效

    • ☐ 骨骼动画、顶点动画

  • 0.0.4 版本：完善空间环境渲染

    • ☐ 体积光/丁达尔效应

    • ☐ 体积雾

    • ☐ 体积云

    • ☐ 雨、雪

    • ☐ 大气散射模型

  • 0.0.5 版本：跨 Python 端 GUI 框架

    • ☐ 支持 PySide

    • ☐ 支持 Tkinter

    • ☐ 支持 PyGame

    • ☐ 支持 PyGTK

  • 0.0.6 版本：跨图形编程接口

    • ☐ 支持 Vulkan

    • ☐ 支持 Direct 3D

    • ☐ 支持 Metal

  • 0.0.7 版本：☐ 支持音效

  • 0.0.8 版本：☐ 支持物理

  • 0.0.9 版本：☐ 支持网络

  • 0.1.0 版本：☐ Glass Engine Editor 编辑器

第三方库引用情况

Glass Engine 基本上完全基于 PyOpenGL 开发，在一些细节方面还引用了一些其他第三方库，现在列举如下：

  • PyOpenGL: 提供渲染的底层接口；

  • PyQt6: 用作 OpenGL 的窗口系统。目前仅支持 PyQt6 窗口系统，对其他 GUI 系统的支持则在进一步的开发计划当中；

  • PyGLM: 用于存储向量、四元数；

  • Numpy: 用于管理连续内存、做矩阵运算等；

  • OpenCV: 用于加载纹理图像、转换图片格式；

  • Pillow: 用于加载一些特殊格式纹理图像；

  • OpenEXR: 用于加载 exr 格式的 HDR 图像；

  • Assimp: 用于加载模型。
